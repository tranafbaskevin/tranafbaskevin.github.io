# KevinOS v2.0

A personal terminal-style portfolio built by Kevin (Trần Bá Kevin).

## Features

- Terminal-inspired interface
- Custom boot screen
- Interactive command system
- Neofetch-style system information
- Command history tracking
- Login history
- Uptime monitor
- Boot instance counter
- Hidden easter eggs
